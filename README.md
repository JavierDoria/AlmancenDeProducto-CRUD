# Sistema de Ventas - ASP.NET

## Descripción
Se creo un sistema para un  almacen  la cual regitrara productos, guardar, editar, actualizar y eliminar.
Tambien se podra colocar la imagen de dicho producto con tan solo colocar la url y todoe sto se guardara en la base de datos

## Tecnologías utilizadas
- ASP.NET Web Forms / MVC
- C#
- SQL Server
- ADO.NET
- HTML5, CSS3
- Bootstrap
